Claude Code セットアップ講座

今日のゴール: Claude Codeを自分のパソコンに入れて、使えるようにする
準備

ターミナルを開く

ターミナル＝パソコンに文字で命令するための画面です。

開き方

  1. キーボードで Cmd ⌘ と スペース を同時に押す
  2. 検索窓が出るので 「ターミナル」 と打つ
  3. 「ターミナル.app」が出てくるので Enter
黒い（または白い）画面が出ればOK。これがターミナルです。
STEP 1

Homebrew を入れる

Homebrewって何？
Macにソフトを入れるための便利ツール。App Storeのターミナル版みたいなもの。

まず、もう入ってないか確認

brew --version
数字が出た人
→ もう入ってるので STEP 2へ
command not found と出た人
→ 下のコマンドで入れる

インストール（入ってなかった人だけ）

下の1行をまるごとコピーして、ターミナルに貼り付けて、Enter。

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

ここでよくあること

「Password:」と出て止まった！
Macのログインパスワードを打ってください。
打っても画面には何も表示されません。見えないけどちゃんと入力できてます。打ったらEnter。
しばらく待つ（3〜10分）
ダウンロード中です。動いてるので放置でOK。

インストールが終わったら

画面の最後の方に 「Next steps:」 と書いてあるはず。
その下に表示されている 2〜3行のコマンドをそのままコピペして実行 してください。

人によって違いますが、だいたいこんな感じ：

echo >> ~/.zprofile echo 'eval "$(/opt/homebrew/bin/brew shellpath)"' >> ~/.zprofile eval "$(/opt/homebrew/bin/brew shellpath)"

確認

brew --version
Homebrew 4.x.x みたいに数字が出ればOK！
STEP 2

node を入れる

Claude Codeを動かすために必要なソフトです。

brew install node

1〜2分で終わります。

確認

node --version
v22.x.x みたいに数字が出ればOK！
STEP 3

VSCode を入れる

VSCodeって何？
プログラマーがよく使うエディタ（メモ帳の超高機能版）。Claude Codeもこの中で使います。

3-1. VSCode本体をインストール

brew install --cask visual-studio-code

1〜3分で終わります。

3-2. VSCodeを開く

code .
「code が見つからない」と出た場合
Cmd ⌘ + スペース → 「Visual Studio Code」と打って直接開いてください。

3-3. Claude Code 拡張機能を入れる

  1. VSCodeの左側にある 四角が4つ並んだアイコン（拡張機能）をクリック
  2. 検索窓に 「Claude Code」 と入力
  3. 「Claude Code」（Anthropic公式）が出てくるので 「Install」 をクリック

3-4. Claude Code をインストール＆起動

VSCode内でターミナルを開きます：

Ctrl + `（バッククォート：数字の1の左のキー）

VSCodeの下半分にターミナルが出ます。ここからはこのターミナルを使います。

Claude Codeをインストール：

npm install -g @anthropic-ai/claude-code

起動：

claude
  1. ブラウザが自動で開く
  2. Anthropicアカウント でログイン（持ってない人はその場で作ってOK）
  3. ログインできたら VSCodeに戻る

動くか試してみよう

Claude Codeの画面に↓を打ってみてください：

今日の日付を教えて
日付が返ってきたら Claude Codeは使えてます！

一旦 Ctrl + C で終了してください。

STEP 4

残りは Claude Code にやってもらう

ここからが本番。自分でコマンドを打つのではなく、AIに「やって」と頼むだけ。

VSCodeのターミナルでもう一度 Claude Code を起動：

claude

起動したら、↓をそのままコピペして送信してください：

Claude Codeに送る以下の3つをやってください。途中で質問せず、全部終わったら結果をまとめて報告してください。 1. GitHub CLI をインストールしてログインする - brew install gh を実行 - gh auth login を実行して、ブラウザでログインするよう案内 - gh auth status で確認 2. Google Cloud CLI をインストールしてセットアップする - brew install google-cloud-sdk を実行 - gcloud init を実行して、ブラウザでGoogleアカウントにログインするよう案内 - プロジェクトは「Create a new project」で「my-tools」という名前で作成 - gcloud auth application-default login を実行 - gcloud services enable sheets.googleapis.com を実行 3. 完了報告 全部終わったら以下の形式で報告： - GitHub CLI: OK / NG - Google Cloud CLI: OK / NG - Google Sheets API: OK / NG
Claude Codeが自動で作業を始めます。
途中でブラウザが開いたら、ログインしてVSCodeに戻ってください。
「Allow」や「Yes」の確認が出たら許可してOKです。
「Permission denied」や「Error」が出た場合
慌てなくてOK。Claude Codeが自動で対処してくれることが多いです。
どうしても進まない場合は手を挙げてください。

全部終わった！おつかれさまでした！

手元で確認してみてください（クリックでチェック）

  • Homebrew brew --version
  • node node --version
  • VSCode 起動できる
  • Claude Code claude --version
  • GitHub gh auth status
  • Google Cloud gcloud auth list
